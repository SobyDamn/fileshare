What's New in v1.1.0

published on dd/mm/yyyy

-Files can be shared in offline mode

Even though in the previous version the files were transferred locally, connection to internet was still required in order to to the service-worker so that client can know the details about the file transfer. In new update options are available to choose between using service worker i.e. using online-mode or do transfer in complete offline mode.
In both modes files are transferred locally, the difference being details of file transferred can be displayed on the page in online-mode.
Check about offline and online modes

-Removed mandotory service worker(now optional).

This will allow files to be transferred without pinging the service worker.

-Option to receive files in offline or online mode.

In offline-mode the file will be saved directly and in online-mode file transfer details will be visible on the browser.
NOTE: File will be received locally and not over the internet.
Know more!

-App theme improved(Dark Theme)

-Multiple receiver can connect to server and request files at once.

This will allow multiple users to connect and receive file at the same time. This will surely reduce the transfer speed for the receiver.

-Multiple files can be received at once

Now receiver can request multiple files in offline-mode.

-Files can also be shared if devices are on same network

In the previous version the receiver was required to connect to the wifi network created by sender i.e. via hotspot, In the new receiver can sender can transfer the files being on the same network i.e. connected with same wifi network.

-Push notification via FCM

New updates notification can be sent directly to user's door.

About

This Android application allows user to share large files to any other system over local network where receiver doesn't require any app to receive.The receiver can use any web browse(currently supporting most of them) and go to the receiving url to receive the file and save them offline. The file transfer happens locally and no Internet connection is required for v1.1.0 or more. This app is made keeping the problem in mind that how time taking it's sometimes to plug the USB cable to transfer the files to the laptop/PC and if we have a broken USB cable like in my case it's more irritating. It's also annoying when we have to share large files with friends and had to decide which app to use as traditional file sharing apps requires the user to use the same application to recieve the files, this app can be helpful in that scenario aswell.
