  • File not saved but shown received keyboard_arrow_right

    This happens when the popup permission is blocked or u had interrupted the process that closes the permission to write the file on your disk ine online-mode.
    To Solve this try below methods
    1.Try enabling popups or remove popup blocker from your browser.
    2.Hard refresh the page or open the site in another window or browser.
    3.Use offline-mode to receive files

  • Not able to connect keyboard_arrow_right

    Make sure that both the sender and receiver is on same network.
    Check here how to receive the files correctly.

  • Getting popups on each receive keyboard_arrow_right

    This can happen if the files are being received in online-mode, as the browser is trying to ping service worker with each file request.
    The pinging the service worker happens using iframe hence the popups.
    You can try using offline-mode

  • Received corrupted files keyboard_arrow_right

    This can happen if the receiving is interrupted somehow try
    1.Asking the sender to send file again.
    2.If above did't work then the file might not be supported by your phone!

  • Notification visible even after closing app keyboard_arrow_right

    This can happen if the app is killed in the background or the app has crashed, to solve this
    Open the app through the notification and the close it again, this will close the app properly.
